The Voice UK: Season 2
Number of YouTube views for each performance of the four finalists.

An experiment in munging data with youtube's api, and then using it to draw some
bar charts with angular.js and raphael.js.

Includes a few (tiny) assets from
http://www.bbc.co.uk/programmes/p012j33g/profiles/teams

A version with data from June 30th, 2013 is here:
  http://www.doc.ic.ac.uk/~tora/voice_2013/page.html
